# Squarespace Style Analyzer Pro – Test Suite

This test suite contains synthetic HTML pages designed to exercise the main
analysis and quality-check features of Squarespace Style Analyzer Pro.

## Structure

- `index.html` – Dashboard linking to all test pages.
- `headings/` – Heading hierarchy edge cases.
- `font-sizes/` – Font size inconsistency and hierarchy tests.
- `images/` – ALT text and image usage tests.
- `buttons/` – Button text and style drift cases.
- `colors/` – Brand palette vs off-brand and overrides.
- `seo/` – SEO metadata and duplication tests.
- `navigation/` – Navigation label vs H1 mismatches and duplicates.

Each page contains a short description at the top of the body explaining what
it is intended to test.
